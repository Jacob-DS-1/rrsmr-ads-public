# Objective 3 Owner 4 System Impact Metrics Runner

This document records the repository-path-aware runner for Objective 3 Owner/Part 4 system impact metrics.

The runner is:

    scripts/run_objective3_system_impact.sh

It consumes the Objective 3 grid master generated by Owner/Part 3 integration:

    data/processed/objective3_smr_integration/grid_master_hourly_2030_2045

The legacy compatibility grid master path is also supported:

    data/processed/objective3_smr_integration/grid_master_hourly_2030_2045.parquet

## Command

Run the integration step first if the grid master is missing:

    scripts/run_objective3_integration.sh

Then run system impact metrics:

    scripts/run_objective3_system_impact.sh --clean

## Generated outputs

Canonical hourly system-impact output:

    data/processed/objective3_smr_integration/system_impact_hourly_2030_2045

Legacy hourly compatibility output:

    data/processed/objective3_smr_integration/system_impact_metrics_hourly_2030_2045.parquet

Annual summary output:

    data/processed/objective3_smr_integration/system_impact_summary_annual_2030_2045.csv

Period summary output:

    data/processed/objective3_smr_integration/system_impact_summary_period_2030_2045.csv

These are generated artifacts and should not be committed.

## Metric definitions

The runner preserves the full grid-master scenario space rather than filtering to average_wind only.

For each hour:

    residual_before_smr_mw = demand_mw - exogenous_supply_mw - imports_net_baseline_mw
    residual_after_smr_mw = residual_before_smr_mw - smr_total_delivered_mw
    gas_needed_before_mw = max(residual_before_smr_mw, 0)
    gas_needed_after_mw = max(residual_after_smr_mw, 0)
    gas_displacement_proxy_mw = gas_needed_before_mw - gas_needed_after_mw
    surplus_after_smr_mw = max(-residual_after_smr_mw, 0)

Low-wind hours are flagged using the 10th percentile of wind_mw across unique generation profiles.

Low-wind support hours are counted where low_wind_flag is true and gas_displacement_proxy_mw is positive.
